# Contributing Guide

## Local setup

1. Copy `.env.example` to `.env.local`.
2. Run `npm install`.
3. Run `npm run dev`.

## Commit style

- Keep commits small and focused.
- Use clear messages (`feat:`, `fix:`, `docs:`, `chore:`).
- Run `npm run lint` before committing.

## Pull request checklist

- [ ] App runs locally
- [ ] Lint passes
- [ ] No secrets added
- [ ] README updated if needed
